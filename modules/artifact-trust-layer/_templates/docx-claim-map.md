# DOCX Claim Map

Use this for memos, LP narratives, board memos, IC memos, diligence summaries, and narrative reports.

| Section | Paragraph Or Heading | Claim Text | Claim Type | Source ID | Evidence Note | Confidence | Issue | Sensitive Language | Review Status |
|---|---|---|---|---|---|---|---|---|---|
|  |  |  | fact/number/date/interpretation/assumption/recommendation/commitment |  |  | high/medium/low/unverified |  | yes/no | pass/warning/blocked |

## Document Review Rules

- Section headings that state conclusions count as claims.
- Narrative interpretation must be labeled as interpretation.
- Sensitive language requires human review.
- Commitments require owner approval.
- Prior memo language is not evidence unless the source packet classifies it as authoritative.
