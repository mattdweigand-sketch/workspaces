# Artifact Boundary

## Platform Owns

- Source files.
- Document storage.
- Permissions and entitlements.
- Version history.
- Delivery.
- Audit trail.
- Figures of record.

## Deterministic Tools Own

- Exact calculations.
- Workbook formulas.
- Tie-outs.
- Duplicate hashes.
- Broken link checks.
- Hidden sheet or range inspection.
- Data freshness checks when the source system exposes dates.

## Kit Owns

- Source authority interpretation.
- Claim and evidence mapping.
- Unsupported claim visibility.
- Stale or superseded source flags.
- Review framing.
- Open confirmation lists.

## Human Owns

- Conflict resolution.
- Final approval.
- External use.
- Decision use.
- Any claim that creates a commitment.

## Hard Rule

If the artifact depends on a figure of record, Kit identifies the source and flags the risk. It does not become the source of truth for the figure.
