# Support

Workspces is maintained as a public toolkit.

## Questions

Use GitHub Issues for:

- setup questions
- unclear architecture guidance
- documentation gaps
- requests for examples

## Bugs

Open an issue with:

- what you tried
- what happened
- what you expected
- which architecture or module was involved
- relevant file paths

Do not include private source material, customer data, credentials, or proprietary documents.

## Private Help

For sensitive implementation questions, contact the maintainer directly rather than posting private context in a public issue.
