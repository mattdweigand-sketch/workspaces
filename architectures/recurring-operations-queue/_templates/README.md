# Response Templates

<!--
This folder holds reusable templates for common request responses.

Four starters ship in this folder. Customize each to your team's voice and
escalation rules; they are drafts, not finished policy:
- acknowledgment.md       Figure / account confirmation drawn from the platform.
- resolution.md           Document re-send or action fulfillment, with the authorization check.
- holding-statement.md    Results answer from reported figures; refers anything forward-looking.
- escalation.md           Confirms receipt, commits the organization to nothing,
                          names who will follow up.

Each starter uses [BRACKET] placeholders for variable content. Figures stay as
placeholders the resolve stage fills from platform-retrieved data, never
hard-coded. Add your own templates for the request types you see most: if you are
writing the same response structure more than three times, it is a template.
-->
