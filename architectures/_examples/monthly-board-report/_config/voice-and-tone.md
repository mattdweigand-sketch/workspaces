(`_shared-config/voice-and-tone.md`). The draft stage reads that for the base voice. This local
file holds only what is specific to the monthly board report, layered on top. L3 reference.

## Team voice
See `_shared-config/voice-and-tone.md` for Northwind's core written voice. Do not duplicate it
here; reference it.

## Board-report voice notes (overlay)
- Write to a board that is numerate and time-poor. Lead with the number and what it means; the
  board does not need the windup.
- Candid about misses. A board forgives a soft month; it does not forgive being managed. Name the
  problem and pair it with an action.
- Confident, not promotional. The metrics carry the message. No "exceptional," "transformative,"
  or similar unless the numbers plainly earn it and the CEO signs off.
- Slightly more formal than internal updates, because this is a governance document, but still
  plain. No jargon a non-operator board member would need decoded.
