# Before You Trust This

The single place that collects every value in this workspace the team has NOT yet confirmed. A
new owner reads it first and clears it before anything decision-bearing ships from the store.

In this worked example the rows are shown **CLEARED**, because the example depicts a running,
validated loop. In a real workspace, these start as `[NEEDS CONFIRMATION]` and a named owner
clears them.
