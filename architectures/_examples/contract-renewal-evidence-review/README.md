# Example: Contract Renewal Evidence Review

Architecture: `evidence-review`

A team has a renewal proposal, prior contract, usage report, and customer email. The workspace reviews what the source set supports before a stakeholder response or decision packet is drafted.

## Boundary

- Document platform owns storage, permissions, version history, and audit.
- Deterministic tools can check exact duplicates.
- Kit owns authority ranking, conflict visibility, and evidence support mapping.
- Human owner resolves conflicts.
