# Stage 02: Message Frame

## Purpose

Frame the communication before drafting.

## Inputs

- `01_verified_facts/output/verified-facts-[name]-[date].md`
- `_config/audience-map.md`
- `_config/voice-and-tone.md`

## Process

1. Identify the audience and desired posture.
2. State the core message.
3. Explain context, impact, and next step.
4. Identify sensitive phrasing.
5. Avoid claims not in the verified facts packet.

## Output

Write to `02_message_frame/output/message-frame-[name]-[date].md`.

## Done Looks Like

The team agrees on what to say, why, and how direct the tone should be.
