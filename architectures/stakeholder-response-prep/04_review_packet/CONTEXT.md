# Stage 04: Review Packet

## Purpose

Prepare the final draft or meeting brief for human review.

## Inputs

- Verified facts.
- Message frame.
- Question prep.
- `_config/before-you-trust-this.md`

## Process

1. Draft the response or meeting brief.
2. Include source notes and open confirmations.
3. Include likely questions when relevant.
4. Name the approver and delivery platform.
5. Do not send externally.

## Output

Write to `04_review_packet/output/review-packet-[name]-[date].md`.

## Done Looks Like

The approver can review, edit, and send through the proper platform.
