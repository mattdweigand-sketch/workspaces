# Stage 03: Question Prep

## Purpose

Prepare likely questions, objections, and approved response posture.

## Inputs

- Verified facts.
- Message frame.
- `_config/audience-map.md`
- `_config/claim-rules.md`

## Process

1. List likely questions.
2. Draft responses grounded in verified facts.
3. Flag questions that require escalation or cannot be answered.
4. Identify recurring concerns worth capturing in memory.
5. Keep sensitive claims within approved boundaries.

## Output

Write to `03_question_prep/output/question-prep-[name]-[date].md`.

## Done Looks Like

The human owner is prepared for the conversation, not just handed a draft.
