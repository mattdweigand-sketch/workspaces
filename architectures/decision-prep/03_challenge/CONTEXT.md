# Stage 03: Challenge

## Purpose

Pressure-test the emerging recommendation.

## Inputs

- `02_options_tradeoffs/output/options-brief-[name]-[date].md`
- `_config/risk-lenses.md`
- `_config/precedent-sources.md`

## Process

1. Identify fragile assumptions.
2. Identify missing evidence.
3. Identify second-order effects.
4. Compare to precedent where available.
5. State what would change the recommendation.

## Output

Write to `03_challenge/output/challenge-memo-[name]-[date].md`.

## Done Looks Like

The decision-maker sees the strongest reasons the recommendation could be wrong.
