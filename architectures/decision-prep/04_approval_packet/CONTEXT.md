# Stage 04: Approval Packet

## Purpose

Prepare the final packet for a human decision.

## Inputs

- Decision frame.
- Options brief.
- Challenge memo.
- `_config/platform-boundary.md`
- `_config/before-you-trust-this.md`

## Process

1. Summarize the decision and recommendation.
2. Include options, tradeoffs, and challenge findings.
3. List unresolved confirmations.
4. Name the platform record and approval owner.
5. State approval conditions and next steps.

## Output

Write to `04_approval_packet/output/approval-packet-[name]-[date].md`.

## Done Looks Like

The human owner can approve, reject, or request more evidence without redoing the analysis.
