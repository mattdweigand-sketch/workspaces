# Stage 02: Options And Tradeoffs

## Purpose

Compare the viable paths before recommending one.

## Inputs

- `01_decision_frame/output/decision-frame-[name]-[date].md`
- `_config/decision-criteria.md`
- Calculations or model output from the deterministic owner, if applicable.

## Process

1. List realistic options, including doing nothing when relevant.
2. Compare upside, downside, risk, timing, and reversibility.
3. Tie every factual claim to its source.
4. Flag missing calculations for the deterministic owner.
5. Do not decide.

## Output

Write to `02_options_tradeoffs/output/options-brief-[name]-[date].md`.

## Done Looks Like

The decision-maker can see the real choices and consequences.
