# Input Types

Define the messy inputs this workspace accepts.

| Input type | Examples | Required metadata | Common risks |
|---|---|---|---|
| Email | Inbound request, forwarded thread | sender, date, thread subject | missing context, stale thread |
| Call notes | Sales call, support call, internal meeting | participants, date, note owner | paraphrase risk |
| File set | PDFs, spreadsheets, screenshots | file name, date, source | stale version, unsupported claim |
| Chat thread | Slack, Teams, SMS | channel, participants, date | informal commitments |
