# Platform Boundary

## Platform Owns

- Final record.
- Workflow state.
- Entitlements and permissions.
- Audit trail.
- Delivery to external systems.

## Workspaces Owns

- Input interpretation.
- Fact and assumption separation.
- Gap detection.
- Confidence notes.
- Human-ready handoff.

## Human Approval

A human approves any record creation, escalation, external send, or irreversible action.
