# Routing Rules

Use deterministic routing where possible.

| Condition | Route |
|---|---|
| Clean record needs to be created | Platform owner |
| Source conflict or missing support | `evidence-review` |
| Human decision needed | `decision-prep` |
| Non-standard case | `exception-handling` |
| Stakeholder communication needed | `stakeholder-response-prep` |
| Lesson should be captured | `institutional-memory-loop` |
