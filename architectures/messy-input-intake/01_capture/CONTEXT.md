# Stage 01: Capture

## Purpose

Gather the input set exactly as it arrived. Do not judge, route, or rewrite yet.

## Inputs

- Original message, call notes, transcript, screenshot, form, or file.
- Source metadata: sender, date, channel, owner, related platform record if one exists.
- `_config/input-types.md`

## Process

1. List every input artifact.
2. Note source, date, channel, and owner where known.
3. Identify whether a platform record already exists.
4. Preserve the sender's words separately from interpreted facts.
5. Flag missing metadata.

## Output

Write to `01_capture/output/input-packet-[name]-[date].md`.

## Done Looks Like

The full input set is captured without interpretation, with missing metadata visible.
