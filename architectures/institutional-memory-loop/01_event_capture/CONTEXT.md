# Stage 01: Event Capture

## Purpose

Capture what happened from source systems and responsible owners.

## Inputs

- Trigger event.
- Platform record or source data.
- Debrief notes.
- `_config/memory-schema.md`

## Process

1. State the event and outcome.
2. List source records and owners.
3. Capture timeline and known facts.
4. Separate facts from reactions.
5. Identify missing context.

## Output

Write to `01_event_capture/output/event-record-[name]-[date].md`.

## Done Looks Like

The factual record is clear enough to interpret without inventing context.
